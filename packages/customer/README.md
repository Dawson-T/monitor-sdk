# `customer`

> 捕获用户信息、业务信息

自动捕获本地存储的用户信息并上报

在 window 上挂载了 `HEIMDALLR_REPORT` 方法，用于上报业务信息

## Options

|配置名称|类型|描述|
|-|-|-|
|customer|Array|客户端上报数据（页面加载完成时收集）|

### customer

|上报字段名称|类型|描述|可选值|
|-|-|-|-|
|name|string|业务字段名称|-|
|postion|string|存储位置|local/session/cookie/global|
